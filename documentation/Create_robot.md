# Создание робота

0. Зайдите в `traffic-editor`, найдите примерные координаты (в метрах), на которых будут стоять роботы и запишите их себе.
1. Название карты на которой будет создан робот
2. Имя робота может быть любым (на английском).
3. Координаты выбирайте на основе координат с карты в traffic-editor.
4. Угол поворота задается в радианах против часовой стрелки.
5. Название зарядной станции указывайте строго по названию точки из `traffic-editor`, которая указана как зарядная станция.
6. Типов робота сейчас(!) всего два: `turtlebot3` и `fast_turtlebot3`.
7. При указании места простоя указывайте названия точек, которые вы указывали в `traffic-editor` как места стоянки. 
8. Тип (park, charge, nothing) зависит от того, хотите ли вы, чтобы роботы куда либо возвращались по окончании задачи (park - стоянка, charge - зарядная станция, nothing - остается на том же месте).
