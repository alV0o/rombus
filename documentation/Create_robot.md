# Создание робота

1. Имя робота может быть любым.
2. Координаты выбирайте на основе координат с карты в traffic-editor (Вы можете открыть его заново и записать их себе).
3. Угол поворота задается в радианах против часовой стрелки.
4. Название зарядной станции указывайте строго по названию точки из `traffic-editor`, которая указана как зарядная станция.
5. Типов робота сейчас(!) всего два: `turtlebot3` и `fast_turtlebot3`.
6. При указании места простоя указывайте названия точек, которые вы указывали в `traffic-editor` как места стоянки. 
7. Тип (park, charge, nothing) зависит от того, хотите ли вы, чтобы роботы куда либо возвращались по окончании задачи (park - стоянка, charge - зарядная станция, nothing - остается на том же месте).
